Scientific Data Engineering Pipeline

Multi-Stage Preprocessing with Polars

To handle the massive Gaia DR3 dataset (originally ~1.8 billion sources), I developed a high-performance preprocessing pipeline using Polars. This pipeline performs rigorous scientific validation, reducing the dataset to 557 million high-confidence stars.

  • Quality Filtering: Implemented filters for Renormalised Unit Weight Error (RUWE < 1.4) and Astrometric Excess Noise (< 1) to ensure high-precision stellar positions.
  • Artifact Removal: Applied a quadratic color-excess mask and filtered out contaminated/blended transits and non-single star candidates to eliminate unreliable data points.
  • Density Analysis: Leveraged HDBSCAN clustering to identify and analyze stellar groupings and density patterns within the galaxy.

Kinematic Feature Engineering

I engineered a simplified stellar kinematic model to enable real-time temporal simulation. By transforming celestial coordinates (RA, Dec, Parallax) into a 3D Cartesian system (x, y, z) and computing velocity vectors (vx, vy, vz) from proper motion and radial velocity, the platform can project star positions forward or backward in time:

Full-Stack System Design

Backend (Cloud-Native API)

The backend is a containerized Python application built with FastAPI and deployed on Google Cloud Run. It pushes the heavy kinematics computation to the BigQuery layer, generating parameterized SQL to ensure both security and performance. The service is stateless, allowing for seamless horizontal scaling.

Frontend (GPU-Accelerated Rendering)

The frontend utilizes React Three Fiber to manage a WebGL-based visualization. By using a single optimized BufferGeometry particle system, the application maintains a buttery-smooth 60 FPS while rendering tens of thousands of stars. Stars are dynamically color-coded by magnitude and include interactive hover states for real-time inspection.

The resulting system allows for a fully interactive exploration of our cosmic neighborhood, bridging the gap between professional astrophysical data and high-performance web visualization.
